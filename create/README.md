# ramdisk.sh
Create a ramdisk and mount it to a location. This should typically have the -- local arg
```bash
RAMDISK="/mnt/RogueCLI_$$/ramdisk/"
rogue ./create/ramdisk.sh 250MB "$RAMDISK" -- local
```
use unmount command to unmount the ramdisk
